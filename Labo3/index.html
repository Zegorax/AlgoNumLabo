Introduction

Le but de ce laboratoire est de développer un programme permettant de résoudre des systèmes d'équations linéaires en JS.
Le défi est d'avoir le temps d'exécution le plus court possible.

Utilisation

Tout d’abord, il faut activer une option de sécurité dans Firefox pour pouvoir lire les fichiers de type JSON sur le disque. Ci-dessous, le lien pour faire cette modification.
Aller à l'option Les trois boutons permettant de (d’essayer) calculer la réponse au système d’équations linéaires proposé par le laboratoire. Le temps de calcul peut être long, car l’opération est répétée 1000 fois (paramétrable) pour calculer une moyenne sur le temps d’exécution. Pour utiliser le programme sur n’importe quel système linéaire :

  • Ajouter le JSON au dossier data
  • Appeler la fonction javascript « compute » en passant le nom de la matrice en paramètre

Fonctionnement

Au lieu de demander les x0 à l'utilisateur, le programme parcourt la fonction sur l'intervalle qui est donné et détecte à quel(s) endroit(s) elle change de signe. Ensuite ces zéros sont envoyés à la fonction qui fait le calcul de la tangente de Newton. Cette fonction va alors calculer la tangente avec les points juste avant ou ces points-là, ce qui va retourner obligatoirement le zéro qui est le plus proche.

Conclusion

On voit que cette méthode d'approximation numérique fonctionne bien car les zéros trouvés sont très proches des zéros réels.
